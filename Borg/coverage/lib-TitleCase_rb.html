 11 # 9c. The first, last, and all major words in titles are capitalized. (See 10a
 12 #  and 16c.)
 13 # 
 14 # In the style favored by the Modern Language Association (MLA) and in that
 15 # recommended by the Chicago Manual of Style (CMS), all words in titles and
 16 # subtitles are capitalized, except articles, coordinating conjunctions,
 17 # prepositions, and the to in infinitives (unless they are the first or last
 18 # word). The articles are a, an, and the; the coordinating conjuctions are and,
 19 # but, for, nor, or, so, and yet. (A list of common prepositions can be found
 20 # in chapter 1; see page 19.) MLA style favors lowercasing all prepositions,
 21 # including long prepositions such as before, between, and through, which
 22 # formerly were capitalized. APA style requires capitalizing any word that has
 23 # four or more letters, including prepositions.
 24 # 
 25 #         * The Scarlet Letter
 26 #         * "How to Be a Leader"
 27 #         * From Here to Eternity
 28 #         * "What This World Is Coming To
 29 #         * Mean Are from Mars, Women Are from Venus [MLA and CMS]
 30 #         * Mean Are From Mars, Women Are From Venus [APA]
 31 # 
 32 # 
 33 # In a title, MLA, APA, and CMS recommend capitalizing all words of a
 34 # hyphenated compound except for articles, coordinating conjunctions, and
 35 # prepositions unless the first element of the compound is a prefix.
 36 # 
 37 #         * "The Building of the H-Bomb" [noun]
 38 #         * "The Arab-Israeli Dilemma" [proper adjective]
 39 #         * "Stop-and-Go Signals" [lowercase coordinating conjunction]
 40 # 
 41 # Because all three style manuals recommend that, in general, compounds with
 42 # prefixes not be hyphenated except in special circumstances, the resulting
 43 # single-word combinations follow the normal rules of capitalization. However,
 44 # if misreading could occur (as in un-ionized or re-cover), if the second
 45 # element begins with a capital letter (pre-Christmas), or if the compound
 46 # results in a doubled letter that could be hard to read (anti-intellectual),
 47 # all style manuals recommend hyphenating the compound. MLA and APA capitalize
 48 # both elements of these compounds with prefixes, whereas CMS capitalizes only
 49 # those elements that are proper nouns or proper adjectives. (While MLA does
 50 # not specifically mention compounds with self-, both APA and CMS usually
 51 # hyphenate self- compounds.)
 52 # 
 53 #         * "Colonial Anti-Independence Poetry" [MLA]
 54 #         * "Anti-Independence Behavior in Pre-Teens" [APA]
 55 #         * "Anti-independence Activities of Delaware's Tories" [CMS]
 56 # 
 57 # 
 58 # However, in titles that appear in lists of references, APA style permits
 59 # capitalizing only the first word and any proper nouns or proper adjectives.

 82   def icap # intelligent capitalization
 83     a = downcase
 84     if a =~ /^['"\(\[']*(\w)/
 85       i = a.index($1)
 86       a[i, 1] = a[i, 1].upcase
 87     end
 88     a
 89   end
 90 
 91   def icap!
 92     replace(icap)
 93   end

 95   def titlecase(exceptions = [])
 96     exclude = EXCEPTIONS + exceptions
 97 #Modification: The original split on word boundaries. We don't like spaces in filenames, so split on _
 98     b = downcase.split(/\b/)
 99     len = b.length
100     b.map_with_index {|w, i|
101       if w.roman_numeral? or (w.length == 1 && b[i+1] == '.')
102         w.upcase
103         # always capitalize first and last words
104       elsif i == 0 or i == len
105         w.icap
106         # don't capitalize the second half of words with apostrophes
107 #Modification: Capitalize if the preceding work ends in a numeric value
108       elsif (exclude.include?(w) or (i>1) && b[i-1] == "'" && b[i-2] =~ /\w/) && (i>1 && b[i-1] =~ /\D\B/)
109         w
110       else
111         w.icap
112       end
113     }.join
114   end
